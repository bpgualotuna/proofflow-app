# ProofFlow - Sistema de Gestión de Pagos con Blockchain

ProofFlow es una aplicación profesional de gestión de pagos que integra tecnología blockchain (Avalanche) para garantizar la integridad y trazabilidad de las transacciones financieras.

## 🚀 Características Principales

- **Dashboard Interactivo**: Visualización en tiempo real de estadísticas de pagos
- **Gestión de Pagos**: Crear, aprobar y rechazar pagos con interfaz intuitiva
- **Verificación Blockchain**: Validación de integridad mediante hashes SHA-256 en Avalanche Fuji C-Chain
- **Material UI**: Interfaz profesional y moderna con componentes de Material-UI
- **Tema Claro/Oscuro**: Soporte completo para modo claro y oscuro
- **Feed en Vivo**: Monitoreo de eventos de blockchain en tiempo real

## 🛠️ Tecnologías

- **Frontend**: React 19 + Vite
- **UI Framework**: Material-UI (MUI) v5
- **Estilos**: Tailwind CSS v4 + CSS personalizado
- **Backend**: Express.js (http://localhost:5000)
- **Blockchain**: Avalanche Fuji C-Chain
- **HTTP Client**: Axios

## 📦 Instalación

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Compilar para producción
npm run build
```

## 🎯 Uso

1. **Crear Pago**: Completa el formulario con monto, descripción y solicitante
2. **Gestionar Pagos**: Aprueba o rechaza pagos pendientes desde la tabla
3. **Verificar Integridad**: Usa el panel de auditoría para verificar hashes en blockchain
4. **Monitorear**: Observa el feed en vivo de eventos de blockchain

## 🔐 Seguridad
